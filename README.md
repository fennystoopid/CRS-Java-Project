# CRS-Java-Project
CRS is a software solution that is designed to help educational institutions to manage how the students recover failed or incomplete courses. A Java-based application is developed using Object-Oriented Programming (OOP) principles to meet that goal.

This project is made via Apache Netbeans, and is needed to run the program.

There are 5 main functionality included in this project:-

1)	User Management:
This is a process of controlling and maintaining users’ accounts and accessibility and interaction with the system.

2)	Course Recovery Plan:
This is a special strategy process that allows students to improve weak performance, through recovering the failed modules with specified timeline and deadlines.

3)	Eligibility Check and Enrolment:
This is a process of checking whether the students meet the required criteria to join the next level of study.

4)	Academic Performance Reporting:
This is the formal document that provides a summary of a student’s progress, achievements, and the recommendations of improvement over a specific period.

5)	Email Notifications:
This is an automated messages send by a system to the users i.e., student’s email address. It helps maintain communication message as alerts, reminders, confirmations and etc between the system and its users to improve accountability.
